I. Overview

Bubble Hub is a web-based milk tea ordering and management platform. It allows users to browse menu items, search and filter by category, customize drinks, manage cart items, and place orders through an online flow.

Customers can complete payment via VNPAY, track order history/detail, and update their profile. Admin users manage products, categories, orders, and users through a dedicated administration area with KPI dashboard.

The project is developed with Java Servlet, JSP, JSTL, and JDBC on SQL Server, following MVC2 architecture using MainController action dispatching for consistent routing.
